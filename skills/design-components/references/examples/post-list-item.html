Post List Item

Two-column row with a date in the left rail and title + summary + meta in the body. Title shifts to accent on hover via the parent group. Used in blog index, home "Latest writing", and the related-posts block on a blog post.

Responsive

The two-column grid (grid-cols-[7rem_1fr]) collapses to a single column below the sm breakpoint (640px). Date renders on its own line above the title; gap-6 shrinks to gap-1 (compact list) or gap-2 (full list) so the date and title still read as one unit, not two stacked blocks.

<li class="py-5 grid
   grid-cols-1 sm:grid-cols-[7rem_1fr]
   gap-1 sm:gap-6
   group cursor-pointer">
  ...
</li>

The "related" variant doesn't use a grid (it's already single-column), so it needs no responsive treatment.
