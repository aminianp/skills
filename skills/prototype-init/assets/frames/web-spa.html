Frame: web-spa

No-op identity frame. The prototype renders directly inside the browser viewport with no device chrome. Use this when targeting responsive web (desktop + mobile breakpoints handled inside the prototype itself, not by an outer frame).

When to use

  • · Building a website, web app, or web SaaS product.
  • · Building a responsive prototype that adapts via media queries inside its own markup.
  • · Default for projects that don’t target a single platform.

Slot

The prototype’s <body> content is the slot. There’s no wrapper to inject into.
